Augmented Reality Product Viewer

For my senior design class I worked in a small team to make an augmented reality app for iOS that allowed potential customers to view products from the company that sponsored our senior project. Unfortunately I am not allowed to post the code online, but my contribution was helping with the app itself in the first semester and setting up the backend of the website interface during the second semester, which is where they uploaded the 3D models and all of the information about them. What this entailed was setting up a Flask server, a Nginx server, a Postgresql server, and an Apollo server, ensuring they could all communicate with one another.


Yelp Database Viewer

For a class we worked in a small group to parse and insert datapoints from Yelp into a database using Postgresql and create an interface for viewing or modifying the data using C# and Windows Forms. You can checkout my code on my github.


Linux Shell Simulator

Transcript Reader

Boolean Decision Diagram Closure

For our algorithms class we had to write a Python script along with a package called pyeda to perform transitive closures on a boolean decision diagram in order to determine if a node could reach another node in an even number of steps. You can checkout my code on my github.
